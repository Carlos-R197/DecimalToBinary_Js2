# Requirements

- The program must be able to convert numbers in base 10 (decimal) to base 2 (binary).

# Acceptance Criteria

- The program must take a decimal number as a parameter and return a number in binary.

# Scenarios

- 0 => 0
- 9 => 1001
- 598 => 1001010110
- 1036 => 10000001100
- 1769 => 11011101001
